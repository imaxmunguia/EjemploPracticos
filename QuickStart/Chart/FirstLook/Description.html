Telerik RadChart is an easy-to-use data presentation component that offers support for all popular chart types. Stunning Vista-style looks, wide set of style presets, and an innovative Chart Wizard for virtually codeless development make it the perfect charting add-on to any application. Supported chart types are: Area, Gantt, Babble, Stacked Area 100%, Spline, StackedSplineArea100, Bar, Line, Point, Stacked Bar, SplineArea, Candlestick, Bezier, Pie, Stacked Area, Stacked Bar 100%, StackedSplineArea.
