This example demonstrates a RadDropDownList with images and multi-line text in its items. Autocomplete is turned on and is set to SuggestAppend. The DblClickRotate property is set to true, so you can navigate to the next item in the DropDownList by double clicking in the text area. In the Configurator you can view and edit properties for the currently selected RadDropDownListItem. Use Add New to create a new item, and Remove to remove the selected item.
