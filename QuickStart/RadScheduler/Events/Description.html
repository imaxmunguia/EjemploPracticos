This example demonstrates the following RadScheduler events: ActiveViewChanging, which fires on view changing; ActiveViewChanged, which fires on view changed; AppointmentMouseDown, which fires on mouse down of an appointment; AppointmentMouseUp, which fires on mouse up of an appointment; AppointmentSelecting, which fires when selecting an appointment; AppointmentSelected, which fires when an appointment is selected; AppointmentEditDialogShowing, which fires when an edit dialog is showing; CellClick, which fires on cell click.
