This example demonstrates the View and Selection events of RadCalendar:
* ViewChanging is a cancelable event that is fired prior to changing the current view to a new one. Its arguments contain a reference to the new view;
* ViewChanged fires when the default view of the calendar is changed;
* SelectionChangingis a cancelable event that is fired prior to adding dates to the SelectedDates collection of RadCalendar;
* SelectionChanged is fired after all selected dates are added to the SelectedDates collection of RadCalendar.
