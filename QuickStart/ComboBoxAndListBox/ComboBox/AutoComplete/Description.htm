RadDropDownList has full autocomplete support - as soon as the user starts typing, the combobox selects or fills-in the first matching entry from the list. The different autocomplete modes can be set through the AutoCompleteMode property and they are:
  • In Suggest mode the first item that starts with the entered text is automatically selected in the dropdown area.
  • In Append mode the first item that starts with the text is automatically appended in the text area, but the drop down area does not appear.
  • In SuggestAppend mode both of the above happen simultaneously. Autocomplete mode can be case-sensitive or case-insensitive
