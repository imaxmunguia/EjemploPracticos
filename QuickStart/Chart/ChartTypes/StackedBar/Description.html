The Stacked Bar Chart displays individual data value contribution of a total across categories. Categories are organized horizontally and values vertically. This chart type is know also as Stacked Column Chart.
The Horizontal Stacked Bar Chart is similar to a stacked bar chart as it also illustrates individual data value contribution of a total across categories. However, categories are organized vertically and values horizontally. In a horizontal stacked bar chart, each data System.Drawing.Point is represented as a horizontal column. The advantage in using a Horizontal Stacked Bar Chart is that the category labels may be easier to read.
Both chart types of are suitable for presenting discrete data.
